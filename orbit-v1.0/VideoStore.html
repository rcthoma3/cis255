Video Store Application

During my time at Delta College, perhaps my most ambitious course project was what I refer to as the "Video Store Application". The application was a database management system built to support the transactional history and inventory maintenance of a fictional video rental store. The system was programmed with Visual Basic, chosen for ease of SQL integration and UI design. Database design had to support the tracking of rental history, transaction history, current rentals, and inventory for several stores within a franchise.

The final product for the database tables were set up as follows:

MoviesStore#(MovieNum, Title, Rating, ReleaseDate, TimesRented)
MovieCopiesStore#(MovieNum, CopyNum, DateReceived, LastHoldDate, IsRented)
MovieRentalsStore#(RentalNum, MovieNum, CopyNum, CustomerNum, DateOfRental, DateOfReturn)
Customers(CustomerNum, FirstName, LastName, StreetAddress, City, State, ZIP9, Phone)
Genre(MovieNum, Genre)


This allowed for extensive tracking of transactional history, current rentals, and inventory. It also allowed for data analytics, such as querying a count of rental items between two dates and delimited by genre, allowing for a "Sales by Genre for the Month" analysis.
